This program tells you how many possibilities were left after each of your Wordle guesses

It compares your guesses to a list of all Wordle answers*
	*many five-letter words may not be included
	*the program only compares to possible Wordle answers

Uses regex in Java to find matches based on current & previous guesses.
Must have a file called "words.txt" in the same directory as "WordleBot.class"
	this file contains all Wordle answers, one on each line
